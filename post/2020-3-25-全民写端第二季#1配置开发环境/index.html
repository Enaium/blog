[全民写端第二季]#1配置开发环境
  1. 下载FoxBase

如果你想修改名字可以在gradle.build文件修改archivesBaseName

  1. 从项目的根目录运行以下命令: ./gradlew setupDevWorkspace idea genIntellijRuns build
  2. 打开 IntelliJ IDEA.
  3. Open Open => Select FoxBase folder
  4. Click Import gradle project
  5. 添加VM options -Dfml.coreMods.load=cn.enaium.foxbase.injection.MixinLoader

完成！
