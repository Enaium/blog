    星露谷模组开发教程#1 配置开发环境

    安装.NET SDK

    首先我们需要下载.NET SDK，这里当前最新的LTS是.NET 8，我们选择.NET 8进行下载。

    点击Download .NET SDK x64进行下载。

    下载完成之后点击安装

    输入命令dotnet --version，如果安装成功你就会看到终端中会输出当前的版本号。

    安装Visual Studio Code

    接着进入Visual Studio Code的下载页面

    选择对应的版本进行下载。

    接受之后一直下一步到完成。

    配置Visual Studio Code

    进入到插件管理搜索C#，点击安装。

    好了，这样我们配置环境这一步就大功告成了。

    安装SMAPI

    这里大家肯定都装了，应该还没有连模组都不会装就来看模组开发了吧。
